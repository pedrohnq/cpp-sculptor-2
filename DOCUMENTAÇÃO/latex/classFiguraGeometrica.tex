\doxysubsection{Detailed Description}
A classe \mbox{\hyperlink{classFiguraGeometrica}{Figura\+Geometrica}} é uma classe base abstrata para representar figuras geométricas. 

A classe \mbox{\hyperlink{classFiguraGeometrica}{Figura\+Geometrica}} serve como uma classe base abstrata para todas as classes que representam figuras geométricas. Ela contém uma função virtual pura \textquotesingle{}draw\textquotesingle{} que deve ser implementada pelas classes derivadas para desenhar a figura no objeto \mbox{\hyperlink{classSculptor}{Sculptor}}.
