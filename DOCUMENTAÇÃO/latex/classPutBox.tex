\doxysubsection{Detailed Description}
A classe \mbox{\hyperlink{classPutBox}{Put\+Box}} representa uma operação de colocação de uma caixa tridimensional no objeto \mbox{\hyperlink{classSculptor}{Sculptor}}. 

A classe \mbox{\hyperlink{classPutBox}{Put\+Box}} é derivada da classe \mbox{\hyperlink{classFiguraGeometrica}{Figura\+Geometrica}} e é usada para adicionar uma caixa tridimensional a um objeto \mbox{\hyperlink{classSculptor}{Sculptor}} especificado por suas coordenadas x, y e z.

Exemplo de uso da classe \mbox{\hyperlink{classPutBox}{Put\+Box}}\+:

\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keywordtype}{int} main() \{}
\DoxyCodeLine{    \textcolor{comment}{// Cria um objeto Sculptor com dimensões 10x10x10}}
\DoxyCodeLine{    \mbox{\hyperlink{classSculptor}{Sculptor}} sculptor(10, 10, 10);}
\DoxyCodeLine{    }
\DoxyCodeLine{    \textcolor{comment}{// Define a cor vermelha como cor padrão}}
\DoxyCodeLine{    sculptor.setColor(1.0, 0.0, 0.0, 1.0);}
\DoxyCodeLine{    }
\DoxyCodeLine{    \textcolor{comment}{// Cria uma caixa vermelha com vértices (1, 1, 1) e (8, 8, 8)}}
\DoxyCodeLine{    \mbox{\hyperlink{classPutBox}{PutBox}} putBox(1, 8, 1, 8, 1, 8, 1.0, 0.0, 0.0, 1.0);}
\DoxyCodeLine{    }
\DoxyCodeLine{    \textcolor{comment}{// Chama a função 'draw' para adicionar a caixa na escultura}}
\DoxyCodeLine{    putBox.draw(sculptor);}

\DoxyCodeLine{    }
\DoxyCodeLine{    \textcolor{keywordflow}{return} 0;}
\DoxyCodeLine{\}}

\end{DoxyCode}
