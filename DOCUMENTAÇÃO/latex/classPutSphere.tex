\doxysubsection{Detailed Description}
A classe \mbox{\hyperlink{classPutSphere}{Put\+Sphere}} representa uma operação de colocação de uma esfera tridimensional no objeto \mbox{\hyperlink{classSculptor}{Sculptor}}. 

A classe \mbox{\hyperlink{classPutSphere}{Put\+Sphere}} é derivada da classe \mbox{\hyperlink{classFiguraGeometrica}{Figura\+Geometrica}} e é usada para adicionar uma esfera tridimensional a um objeto \mbox{\hyperlink{classSculptor}{Sculptor}} especificado por seu centro (x, y, z) e raio.

Exemplo de uso da classe \mbox{\hyperlink{classPutSphere}{Put\+Sphere}}\+: 
\begin{DoxyCode}{0}

\DoxyCodeLine{   \mbox{\hyperlink{classSculptor}{Sculptor}} sculptor(10, 10, 10);}
\DoxyCodeLine{   }
\DoxyCodeLine{   \textcolor{comment}{// Cria uma esfera vermelha com raio 3 no centro da escultura}}
\DoxyCodeLine{   \mbox{\hyperlink{classPutSphere}{PutSphere}} putSphere(5, 5, 5, 3, 1.0, 0.0, 0.0, 1.0);}
\DoxyCodeLine{   }
\DoxyCodeLine{   \textcolor{comment}{// Chama a função 'draw' para adicionar a esfera na escultura}}
\DoxyCodeLine{   putSphere.draw(sculptor);}
\DoxyCodeLine{   }
\DoxyCodeLine{   sculptor.writeOFF(\textcolor{stringliteral}{"{}escultura.off"{}});}
\DoxyCodeLine{   }
\DoxyCodeLine{   \textcolor{keywordflow}{return} 0;}

\end{DoxyCode}
